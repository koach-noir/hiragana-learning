# 🛠️ 画像ダウンロードツール

いらすとやから画像を自動ダウンロードするツールです。

## 📁 ファイル構成

```
tools/
├── README.md              # このファイル
├── image-candidates.csv   # 画像候補URL一覧
├── download_images.py     # ダウンロードスクリプト
└── downloads/             # ダウンロード先（自動生成）
    ├── e/
    │   ├── e_01.png
    │   ├── e_02.png
    │   ├── ehon_01.png
    │   └── ...
    └── no/
        ├── nori_01.png
        └── ...
```

## 🚀 使い方

### 1. 必要なパッケージをインストール

```bash
pip install requests beautifulsoup4
```

### 2. tools フォルダに移動

```bash
cd tools
```

### 3. スクリプト実行

```bash
python download_images.py
```

### 4. 出力確認

- `downloads/` フォルダに画像が保存されます
- `download_report.csv` に結果が記録されます

## 🔍 選定作業

ダウンロード後:

1. `downloads/` フォルダを開く
2. 各単語につき **1つ** を選ぶ
3. 選んだ画像を `../images/[ひらがな]/` にコピー
4. `../data/words.js` の `image` パスを更新

### 選定基準

- ✅ **シンプル**: 幼児が認識しやすい
- ✅ **明確**: 何を表しているか分かりやすい
- ✅ **カラフル**: 興味を引く色使い
- ❌ **複雑**: 情報量が多すぎる
- ❌ **抽象的**: 何か分かりにくい

## ⚠️ 注意事項

- いらすとやの利用規約を遵守してください
- 1つの制作物につき20点まで無料利用可能
- 商用利用の場合は規約を確認してください
- サーバー負荷軽減のため、スクリプトは1秒間隔でリクエストします

## 🔧 トラブルシューティング

### 「画像URL取得失敗」が出る場合

- ページURLが変更されている可能性があります
- 手動でいらすとやを検索して、正しいURLに更新してください

### 「ダウンロードエラー」が出る場合

- ネットワーク接続を確認してください
- 時間をおいて再実行してください
